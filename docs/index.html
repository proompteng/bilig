@bilig/headless

bilig Spreadsheet logic for Node services.

When a feature still depends on formulas, build a WorkPaper in TypeScript, write the inputs, read the result, and save the workbook as JSON.

Use it when

Your spreadsheet is already part of the product.

Quotes, budgets, commissions, and imports often start as spreadsheet models. bilig is for the point where that model has to run in a service or an agent tool.

01 / model

Start with arrays, JSON records, or workbook data.

Use formulas where a formula is still the clearest way to express the rule.

02 / edit

Write the cell you intend to change.

No screen scraping and no invisible browser session sitting behind the API.

03 / check

Read the value after recalculation.

Return the number your code actually observed.

04 / keep

Persist the WorkPaper document.

Store the workbook as JSON and restore the same formulas later.

eval.ts TypeScript 
import {
  WorkPaper,
  createWorkPaperFromDocument,
  exportWorkPaperDocument,
  parseWorkPaperDocument,
  serializeWorkPaperDocument,
} from "@bilig/headless";

type NumericCell = { value: number };

const workbook = WorkPaper.buildFromSheets({
  Revenue: [
    ["Region", "Customers", "ARPA", "Revenue"],
    ["West", 20, 1200, "=B2*C2"],
    ["East", 30, 250, "=B3*C3"],
    ["Central", 18, 300, "=B4*C4"],
  ],
  Summary: [
    ["Metric", "Value"],
    ["Total revenue", "=SUM(Revenue!D2:D4)"],
  ],
});

function numberValue(cell: unknown): number {
  if (typeof cell === "object" && cell !== null && typeof (cell as NumericCell).value === "number") {
    return (cell as NumericCell).value;
  }

  throw new Error(`Expected numeric cell value, got ${JSON.stringify(cell)}`);
}

const summary = workbook.getSheetId("Summary");
const revenue = workbook.getSheetId("Revenue");
if (summary === undefined || revenue === undefined) {
  throw new Error("Missing sheet");
}

const before = numberValue(workbook.getCellValue({ sheet: summary, row: 1, col: 1 }));
workbook.setCellContents({ sheet: revenue, row: 1, col: 1 }, 32);

const saved = serializeWorkPaperDocument(exportWorkPaperDocument(workbook, { includeConfig: true }));
const restored = createWorkPaperFromDocument(parseWorkPaperDocument(saved));
const restoredSummary = restored.getSheetId("Summary");
if (restoredSummary === undefined) {
  throw new Error("Missing restored Summary sheet");
}

const after = numberValue(restored.getCellValue({ sheet: restoredSummary, row: 1, col: 1 }));
console.log({ before, after, bytes: saved.length, verified: before === 36900 && after === 51300 });

Put WorkPaper behind a route, a job, or an agent tool.
