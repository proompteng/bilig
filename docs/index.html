headless workbook runtime

bilig runs spreadsheet logic without the spreadsheet ui.

a TypeScript spreadsheet engine for Node services, coding agents, local-first automation, formulas, structural edits, persistence, and benchmarkable workbook workflows.

use it when a grid is the wrong abstraction.

bilig exposes workbook behavior as a runtime API, so services and agents can work with formulas and workbook state directly.

formulas and workbook edits

create sheets, set values, evaluate formulas, move cells, undo, redo, snapshot, restore, and inspect workbook state.

agent-addressable runtime

stable request and document shapes let coding agents mutate spreadsheet state without screen scraping a browser grid.

local-first persistence

round-trip workbook documents through explicit serialization and restoration helpers built for deterministic service workflows.

quickstart

install the package and build a workbook from plain arrays.

pnpm add @bilig/headless
import { WorkPaper } from "@bilig/headless";

const workbook = WorkPaper.buildFromSheets({
  Sheet1: [
    [10, 20, "=A1+B1"],
    [7, "=A2*3", null],
  ],
});

const sheet = workbook.getSheetId("Sheet1");
console.log(workbook.getCellValue({ sheet, row: 0, col: 2 }));
