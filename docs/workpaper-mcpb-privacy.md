# Bilig WorkPaper MCPB Privacy Policy

This privacy policy applies to the **Bilig WorkPaper** Claude Desktop MCPB
extension generated by this repository.

## Data collection

The extension runs locally on your computer through Claude Desktop stdio. The
Bilig WorkPaper extension does not collect telemetry, account identifiers,
analytics events, workbook contents, formulas, calculated cell values, prompts,
or generated WorkPaper JSON files for Proompt Engineering.

## Usage and storage

On first run, the extension creates a local `workpaper.json` file inside the
installed extension directory if one does not already exist. When you call
`set_cell_contents`, the extension updates that local WorkPaper JSON file so
Claude can verify recalculated readback and persistence.

The extension does not upload that file to Proompt Engineering. Claude Desktop
and Anthropic may process your prompts, tool calls, tool results, uploaded files,
and related usage data according to Anthropic's policies and the settings for
your Claude account or organization.

## Third-party sharing

The extension does not share data with Proompt Engineering or any service
operated by Proompt Engineering. It does not open network connections for
Proompt Engineering services.

The extension is used inside Claude Desktop, so data you choose to send to
Claude is handled by Anthropic. Review Anthropic's privacy policy and your
Claude plan or organization settings for that processing.

## Data retention

Proompt Engineering does not retain data from this local extension because the
extension does not transmit data to Proompt Engineering.

The local `workpaper.json` file remains on your computer until you delete the
extension, delete the installed extension data, or replace the file yourself.
Anthropic retention is governed by Anthropic's policies and your Claude account
or organization settings.
