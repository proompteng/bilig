# @bilig/crdt

Deterministic CRDT batch and replica primitives for bilig workbook synchronization.

## Install

```bash
npm install @bilig/crdt
```

## Package entrypoints

- ESM: `./dist/index.js`
- Types: `./dist/index.d.ts`

This package is part of the [bilig](https://github.com/proompteng/bilig) monorepo.
