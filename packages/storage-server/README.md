# @bilig/storage-server

Server-side storage abstractions for bilig snapshots, logs, and sync persistence.

## Install

```bash
npm install @bilig/storage-server
```

## Package entrypoints

- ESM: `./dist/index.js`
- Types: `./dist/index.d.ts`

This package is part of the [bilig](https://github.com/proompteng/bilig) monorepo.
