# @bilig/storage-browser

Browser persistence primitives for bilig snapshots, queues, and local-first state.

## Install

```bash
npm install @bilig/storage-browser
```

## Package entrypoints

- ESM: `./dist/index.js`
- Types: `./dist/index.d.ts`

This package is part of the [bilig](https://github.com/proompteng/bilig) monorepo.
