# @bilig/worker-transport

Worker transport client and host primitives for bilig browser runtime execution.

## Install

```bash
npm install @bilig/worker-transport
```

## Package entrypoints

- ESM: `./dist/index.js`
- Types: `./dist/index.d.ts`

This package is part of the [bilig](https://github.com/proompteng/bilig) monorepo.
