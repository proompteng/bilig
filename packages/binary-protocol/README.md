# @bilig/binary-protocol

Binary frame encoder and decoder for bilig sync, agent, and local transport.

## Install

```bash
npm install @bilig/binary-protocol
```

## Package entrypoints

- ESM: `./dist/index.js`
- Types: `./dist/index.d.ts`

This package is part of the [bilig](https://github.com/proompteng/bilig) monorepo.
