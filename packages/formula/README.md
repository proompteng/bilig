# @bilig/formula

Excel-style formula parsing, binding, compilation, and evaluation for bilig.

## Install

```bash
npm install @bilig/formula
```

## Package entrypoints

- ESM: `./dist/index.js`
- Types: `./dist/index.d.ts`

This package is part of the [bilig](https://github.com/proompteng/bilig) monorepo.
