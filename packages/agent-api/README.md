# @bilig/agent-api

Binary agent API frames and typed workbook session operations for bilig.

## Install

```bash
npm install @bilig/agent-api
```

## Package entrypoints

- ESM: `./dist/index.js`
- Types: `./dist/index.d.ts`

This package is part of the [bilig](https://github.com/proompteng/bilig) monorepo.
