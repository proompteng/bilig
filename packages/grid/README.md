# @bilig/grid

Excel-like React grid shell components for bilig built on Glide Data Grid.

## Install

```bash
npm install @bilig/grid
```

## Package entrypoints

- ESM: `./dist/index.js`
- Types: `./dist/index.d.ts`

This package is part of the [bilig](https://github.com/proompteng/bilig) monorepo.
