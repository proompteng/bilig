# @bilig/protocol

Shared protocol types, enums, opcodes, and spreadsheet limits for bilig.

## Install

```bash
npm install @bilig/protocol
```

## Package entrypoints

- ESM: `./dist/index.js`
- Types: `./dist/index.d.ts`

This package is part of the [bilig](https://github.com/proompteng/bilig) monorepo.
